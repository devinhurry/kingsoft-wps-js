# Structure Examples

This section contains examples of some of the most commonly used data
structures in MS-DOC files. The examples are based on common
computational tasks.

Section [3.1](#Section_a30a9a42cf784974b99fd559639ee383) provides
examples of the data structures that are used to find the text of the
document.

Section [3.2](#Section_6777a859d537486b96c7fb0c959a5bf9) provides
examples of the data structures that are used to find the properties of
a [**section**](#gt_49a2b98a-d101-4889-9108-87f567e758cf), including
page size and margins.

Section [3.3](#Section_b5e6b16c4c094eda81bb9757385dc9ba) provides
examples of the data structures that are used to determine the ranges of
[**bookmarks**](#gt_42f9c2f4-8a4b-4d64-a0e1-fc071debdf4c).

Sections [3.4](#Section_5782a6df448e48b9a7887518a0372070) and
[3.5](#Section_0471d9a7247544e9a60d70e2a9af1b24) provide examples of the
data structures that are used to determine direct character and
paragraph formatting.

Section [3.6](#Section_efb41f4ca92042d5860f850266e0b55b) provides an
example of the use of the [sprmTInsert](#table-properties) value to
define a table row and of the use of an [**ItcFirstLim**](#itcfirstlim)
structure to specify a range of cells to which a [**Sprm**](#sprm)
structure applies.

Finally, Section [3.7](#Section_39d71c1c1ed04cbe8aad314d716f9649)
provides an example of the data structures that are used to determine
the formatting and [**number
text**](#gt_f0fa9b31-039b-4e61-ae7f-dd2a397a2f68) of a list.
