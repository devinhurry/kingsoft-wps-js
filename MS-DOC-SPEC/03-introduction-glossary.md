# Introduction

## Glossary

This document uses the following terms:

> <span id="gt_d61b4a07-b6c1-4c08-9ebd-1d00b360f953"
> class="anchor"></span>**accelerator key**: Any combination of keys
> that are pressed simultaneously to run a command.
>
> <span id="gt_390da589-0ab2-467d-a90e-69ca94f8c2f4"
> class="anchor"></span>**allocated command**: A built-in command that
> requires the user to specify a value for a parameter when customizing
> the command.
>
> <span id="gt_084d6638-17a5-4bf5-8bf1-70881bdeb997"
> class="anchor"></span>**anchor**: A set of qualifiers and quantifiers
> that specifies the location of an element or object within a document.
> These values are typically relative to another element or known
> location in the document, such as the edge of a page or margin.
>
> <span id="gt_0a40ee6b-0144-44fd-b4de-d3f1aa29d008"
> class="anchor"></span>**annotation bookmark**: An entity in a document
> that is used to denote the range of content to which a comment
> applies.
>
> <span id="gt_79fa85ca-ac61-467c-b819-e97dc1a7a599"
> class="anchor"></span>**ASCII**: The American Standard Code for
> Information Interchange (ASCII) is an 8-bit character-encoding scheme
> based on the English alphabet. ASCII codes represent text in
> computers, communications equipment, and other devices that work with
> text. ASCII refers to a single 8-bit ASCII character or an array of
> 8-bit ASCII characters with the high bit of each character set to
> zero.
>
> <span id="gt_24ddbbb4-b79e-4419-96ec-0fdd229c9ebf"
> class="anchor"></span>**Augmented Backus-Naur Form (ABNF)**: A
> modified version of Backus-Naur Form (BNF), commonly used by Internet
> specifications. ABNF notation balances compactness and simplicity with
> reasonable representational power. ABNF differs from standard BNF in
> its definitions and uses of naming rules, repetition, alternatives,
> order-independence, and value ranges. For more information, see
> [\[RFC5234\]](https://go.microsoft.com/fwlink/?LinkId=123096).
>
> <span id="gt_8e3c4123-5325-4883-8195-ddb4531957fe"
> class="anchor"></span>**auto spacing**: A condition in which space is
> inserted automatically before and after a series of consecutive
> paragraphs that do not have breaks or other items between them.
>
> <span id="gt_7df2d57b-8f21-4a9f-8f78-5186d57f09f3"
> class="anchor"></span>**AutoCaption**: A feature that adds a
> [**caption**](#gt_81d81412-1575-4084-ba61-742de406b418) to an object
> automatically when the object is inserted in a document.
>
> <span id="gt_89235fe3-510c-48d6-bba9-bf729ca28ae4"
> class="anchor"></span>**AutoCorrect**: A feature that corrects errors
> and makes other substitutions in a document automatically by using
> default and user-defined settings.
>
> <span id="gt_e6b48619-23ab-4988-a50b-e19e60cfb108"
> class="anchor"></span>**auto-hyphenated**: A condition of content
> where the distance between the text is measured and maintained to
> force breaks automatically in elongated words that would not otherwise
> end correctly on a line.
>
> <span id="gt_6c67a08d-5bf2-4f03-b2cb-83e29db2778c"
> class="anchor"></span>**automark file**: A file that stores the text,
> location, and index level of a set of characters that were marked for
> inclusion in a document index.
>
> <span id="gt_f4f3be71-a6a0-43a1-974d-cf345372f5bf"
> class="anchor"></span>**AutoSummary**: A process in which key points
> are identified in selected text by analyzing document content. A score
> is assigned to each sentence; sentences that contain frequently used
> words are given a higher score.
>
> <span id="gt_90dc5ce2-c9bd-43be-8104-935d905c2a7a"
> class="anchor"></span>**AutoText**: A storage location for text and
> graphics, such as a standard contract clause, that can be used
> multiple times in one or more documents. Each selection of text or
> graphics is recorded as an AutoText entry and assigned a unique name.
>
> <span id="gt_cbe82fe4-5cf4-42c3-8564-bd9cbd2a5482"
> class="anchor"></span>**bar tab**: A tab that specifies where to draw
> a vertical line or bar in a paragraph. It neither affects the position
> of characters nor creates a custom tab stop in a paragraph.
>
> <span id="gt_29ba9472-7f9d-4dc6-b675-98e390c865ca"
> class="anchor"></span>**bidirectional compatibility**: The ability to
> display and process text in two directions, right-to-left and
> left-to-right.
>
> <span id="gt_6f6f9e8e-5966-4727-8527-7e02fb864e7e"
> class="anchor"></span>**big-endian**: Multiple-byte values that are
> byte-ordered with the most significant byte stored in the memory
> location with the lowest address.
>
> <span id="gt_42f9c2f4-8a4b-4d64-a0e1-fc071debdf4c"
> class="anchor"></span>**bookmark**: An entity that is used in a
> document to denote the beginning and ending character positions of
> specific text in the document, and optionally, metadata about that
> text or its relationship to other referenced parts of the document.
>
> <span id="gt_81d81412-1575-4084-ba61-742de406b418"
> class="anchor"></span>**caption**: One or more characters that can be
> used as a label for display purposes or as an identifier.
>
> <span id="gt_22155692-26b0-4146-9737-ef43f4b31305"
> class="anchor"></span>**cascading style sheet (CSS)**: An extension to
> [**HTML**](#gt_549c4960-e8be-4c24-bc2b-b86530f1c1bf) that enables
> authors and users of HTML documents to attach style sheets to those
> documents, as described in
> [\[CSS-LEVEL1\]](https://go.microsoft.com/fwlink/?LinkId=99527) and
> [\[CSS-LEVEL2\]](https://go.microsoft.com/fwlink/?LinkId=114090). A
> style sheet includes typographical information about the appearance of
> a page, including the font for text on the page.
>
> <span id="gt_43d1e51e-4f26-493b-b7c9-e84e920d7461"
> class="anchor"></span>**cell**: A box that is formed by the
> intersection of a row and a column in a worksheet or a table. A cell
> can contain numbers, strings, and formulas, and various formats can be
> applied to that data.
>
> <span id="gt_30232e45-c2f5-4d92-854c-74ffdea1e163"
> class="anchor"></span>**cell margin**: A measurement of the distance
> between the border of a cell and the nearest pixel in a character or
> digit of data in the cell. There are top, bottom, right, and left
> margins. See also [**cell
> spacing**](#gt_4a64964b-c1e8-41b3-b3da-e9866f5227d2).
>
> <span id="gt_4a64964b-c1e8-41b3-b3da-e9866f5227d2"
> class="anchor"></span>**cell spacing**: A measurement of the distance
> between the cells of a table or worksheet. Most tables and worksheets
> are implemented with contiguous cells, in which case the cell spacing
> value is 0 (zero). See also [**cell
> margin**](#gt_30232e45-c2f5-4d92-854c-74ffdea1e163).
>
> <span id="gt_28faddfb-e8ce-4bde-8e29-a85ce2dd5e49"
> class="anchor"></span>**CGAPI**: An API that is implemented by
> [**grammar checkers**](#gt_19363950-99e9-4f11-a562-96e4dd4ea5ce) that
> have been licensed to Microsoft Corporation by external vendors.
>
> <span id="gt_1831737e-39f6-45f6-a116-89767cc45f98"
> class="anchor"></span>**chapter numbering**: A page numbering format
> in which pages are numbered relative to the beginning of a chapter
> within a document instead of the beginning of the document. The
> chapter number is typically included in a page number; for example "3
> – 2," where "3" is the chapter number and "2" is the number of that
> page within that chapter.
>
> <span id="gt_44c69cba-79c8-4147-8b3a-ead7b39a63d5"
> class="anchor"></span>**character pitch**: A quality that measures the
> number of characters that can be printed in a horizontal inch. Pitch
> is typically used to measure monospace fonts.
>
> <span id="gt_5004b992-4a9c-41c9-b65c-b2e7a2b04204"
> class="anchor"></span>**character set**: A mapping between the
> characters of a written language and the values that are used to
> represent those characters to a computer.
>
> <span id="gt_f15b67ae-3a41-4195-9fac-486a7c1dfc8a"
> class="anchor"></span>**character unit**: A horizontal unit of
> measurement that is relative to the document grid and is used to
> position content in a document.
>
> <span id="gt_e433c806-6cb6-46a2-bb95-523df8818c99"
> class="anchor"></span>**class identifier (CLSID)**: A GUID that
> identifies a software component; for instance, a DCOM object class or
> a COM class.
>
> <span id="gt_210637d9-9634-4652-a935-ded3cd434f38"
> class="anchor"></span>**code page**: An ordered set of characters of a
> specific script in which a numerical index (code-point value) is
> associated with each character. Code pages are a means of providing
> support for [**character
> sets**](#gt_5004b992-4a9c-41c9-b65c-b2e7a2b04204) and keyboard layouts
> used in different countries/regions. Devices such as the display and
> keyboard can be configured to use a specific code page and to switch
> from one code page (such as the United States) to another (such as
> Portugal) at the user's request.
>
> <span id="gt_ef2ebebc-1760-407a-9ace-af48f9050e02"
> class="anchor"></span>**Component Object Model (COM)**: An
> object-oriented programming model that defines how objects interact
> within a single process or between processes. In
> [**COM**](#gt_ef2ebebc-1760-407a-9ace-af48f9050e02), clients have
> access to an object through interfaces implemented on the object. For
> more information, see
> [\[MS-DCOM\]](%5bMS-DCOM%5d.pdf#Section_4a893f3dbd2948cd9f43d9777a4415b0).
>
> <span id="gt_03a9d0ca-2f10-4f3d-b910-052714a96f7d"
> class="anchor"></span>**connection string**: A series of arguments,
> delimited by a semicolon, that defines the location of a database and
> how to connect to it.
>
> <span id="gt_6ee69c13-ffa9-43b1-989e-a942dceb1fa5"
> class="anchor"></span>**custom toolbar**: A type of toolbar that
> contains a user-defined set of controls and is not included in an
> application by default. A custom toolbar has a toolbar identifier
> value of "1".
>
> <span id="gt_75556eac-bd5a-4dc2-b7b5-48c27bfa1c93"
> class="anchor"></span>**custom toolbar control**: A user-defined
> control that can be added to a toolbar. A custom toolbar control has a
> [**toolbar control identifier
> (TCID)**](#gt_28d9adbd-b340-469a-97cd-3befd6a782b3) value of "1" and
> can be one of the following types of controls: ActiveX, Button,
> ComboBox, DropDown, Edit, or Popup.
>
> <span id="gt_c2ecfc06-8cce-4183-88d4-b34aaea78850"
> class="anchor"></span>**deletion point**: A position between two
> existing characters, or a position before or after a character, where
> text was removed. If a caret is positioned at a deletion point, the
> point can retain unique formatting and that formatting can be
> reapplied to any text that is inserted at the deletion point.
>
> <span id="gt_ad0cf6e3-05c3-482d-ab0f-617f91871189"
> class="anchor"></span>**digital signature**: A value that is generated
> by using a digital signature algorithm, taking as input a private key
> and an arbitrary-length string, such that a specific verification
> algorithm is satisfied by the value, the input string, and the public
> key corresponding to the input private key.
>
> <span id="gt_7b9a05f4-888b-4176-b00a-115046299e1b"
> class="anchor"></span>**document**: An object in a content database
> such as a file, folder, list, or site. Each object is identified by a
> [**URI**](#gt_e18af8e8-01d7-4f91-8a1e-0fb21b191f95).
>
> <span id="gt_f1f5f018-eebc-4631-9036-ed857713c71c"
> class="anchor"></span>**document grid**: A feature that enables the
> precise layout of full-width East Asian language characters by
> specifying the number of characters per line and the number of lines
> per page.
>
> <span id="gt_20e70600-75a3-424c-b1ae-ce3b9f6047ff"
> class="anchor"></span>**document template**: A file that serves as the
> basis for new documents.
>
> <span id="gt_8dcae18f-67a0-4282-860e-1b6713fe6aae"
> class="anchor"></span>**East Asian character**: A character that is
> part of the Simplified Chinese, Traditional Chinese, Japanese, or
> Korean [**character set**](#gt_5004b992-4a9c-41c9-b65c-b2e7a2b04204).
>
> <span id="gt_12f63b8b-1c85-4855-9ae1-e6b05720bcfc"
> class="anchor"></span>**East Asian language**: A spoken or written
> communication that consists of words that are used within the
> grammatical and syntactic structure of Simplified Chinese, Traditional
> Chinese, Japanese, or Korean.
>
> <span id="gt_75586443-d3aa-4e41-a68f-f816ee4137d8"
> class="anchor"></span>**East Asian line breaking rules**: A set of
> algorithms that define how text is parsed and displayed to ensure that
> line breaks and word wraps follow the rules of various East Asian
> languages, including Simplified Chinese, Traditional Chinese,
> Japanese, and Korean.
>
> <span id="gt_2dfdaa55-39fd-4dc0-a630-dae1c4c09c9b"
> class="anchor"></span>**end of cell mark**: A character with a
> hexadecimal value of "0x07" that is used to indicate the end of a cell
> in a table.
>
> <span id="gt_9fe5b7d0-b009-4e3b-9368-10a2a3ab8da9"
> class="anchor"></span>**end of row mark**: The combination of a
> character, hexadecimal value of "0x07", and a paragraph property,
> sprmPFTtp, that is used to indicate the end of a row in a table.
>
> <span id="gt_d14d8e59-0dcc-499b-b57f-0c48f0420137"
> class="anchor"></span>**endnote**: A note that appears at the end of a
> section or document and that is referenced by text in the main body of
> the document. An endnote consists of two linked parts, a reference
> mark within the main body of text and the corresponding text of the
> note.
>
> <span id="gt_253d5240-eccc-4f28-a27b-1ff572f79b68"
> class="anchor"></span>**endnote continuation notice**: A set of
> characters indicating that an endnote continues to the next page. The
> default notice is blank.
>
> <span id="gt_9f358e09-3d87-49bc-8818-d328793e9ef5"
> class="anchor"></span>**endnote continuation separator**: A set of
> characters that indicates the end of document text on a page and the
> beginning of endnotes that continue from the preceding page.
>
> <span id="gt_d30c582e-70a2-419b-ba89-59fc910d165b"
> class="anchor"></span>**endnote separator**: A set of characters that
> separates document text from endnotes about that text. The default
> separator is a horizontal line.
>
> <span id="gt_f819dd42-7f44-4613-8231-d5ad47f2bbcc"
> class="anchor"></span>**field**: An element or attribute in a data
> source that can contain data.
>
> <span id="gt_1be27f15-12a7-416d-9db9-8dc1b8cf78a3"
> class="anchor"></span>**field type**: A name that identifies the
> action or effect that a field has within a document. Examples of field
> types are Author, Page, Comments, and Date.
>
> <span id="gt_f2bf797b-e733-4fb9-b5e5-7e122f4abbe0"
> class="anchor"></span>**file allocation table (FAT)**: A data
> structure that the operating system creates when a volume is formatted
> by using [**FAT**](#gt_f2bf797b-e733-4fb9-b5e5-7e122f4abbe0) or FAT32
> file systems. The operating system stores information about each file
> in the [**FAT**](#gt_f2bf797b-e733-4fb9-b5e5-7e122f4abbe0) so that it
> can retrieve the file later.
>
> <span id="gt_c5883e8c-12bd-4d69-b0d0-6588eec448cc"
> class="anchor"></span>**footer**: One or more lines of text in the
> bottom margin area of a page in a document or a slide in a
> presentation. A footer typically contains elements such as the page
> number and the name of the file.
>
> <span id="gt_17b5b16e-d079-48c9-a5d1-db80406c45a8"
> class="anchor"></span>**footnote**: A note that appears at the end of
> a page, section, chapter, or publication. It explains, comments on, or
> provides references for text in the main body of a document. A
> footnote consists of two linked parts, a reference mark within the
> main body of the document and the corresponding text of the note.
>
> <span id="gt_fe7959b9-9b77-4bc0-887c-062cdedb9f68"
> class="anchor"></span>**footnote continuation notice**: A set of
> characters indicating that a footnote continues to the next page. The
> default notice is blank.
>
> <span id="gt_1650f6e9-5799-4a9d-895a-8a4107009dc7"
> class="anchor"></span>**footnote continuation separator**: A set of
> characters that indicates the end of document text on a page and the
> beginning of footnotes that continue from the preceding page.
>
> <span id="gt_8504ae38-cbfd-4547-a410-03a7629920ae"
> class="anchor"></span>**footnote separator**: A set of characters that
> separates document text from footnotes about that text. The default
> separator is a horizontal line.
>
> <span id="gt_8fa0b648-6259-4969-8561-a87b325fb240"
> class="anchor"></span>**form field**: A data-entry area on a webpage,
> document, or form.
>
> <span id="gt_a2a42bb4-740d-4d06-9922-2e10d6233622"
> class="anchor"></span>**format consistency checker**: An application
> that applies a wavy blue underline to text where the formatting is
> similar, but not identical, to comparable text in a document.
>
> <span id="gt_2f3fbf82-4359-4b12-aeec-1968eb797ee5"
> class="anchor"></span>**format consistency-checker bookmark**: An
> entity in a document that is used to denote text where the formatting
> is similar, but not identical, to comparable text in the document, and
> the user indicated that the formatting inconsistency is not to be
> flagged.
>
> <span id="gt_1c54183d-33fa-463c-b929-e996d716e3bb"
> class="anchor"></span>**frame**: A space, displayed onscreen as a box,
> that contains a specific element of a publication.
>
> <span id="gt_83dde85c-7bbe-448d-8fc9-ef0a5c7f7a31"
> class="anchor"></span>**full save**: A process in which an existing
> file is overwritten with all of the additions, changes, and other
> content in a document.
>
> <span id="gt_20327411-614f-448f-aa96-4c96726830eb"
> class="anchor"></span>**full screen view**: A document view that
> expands the display of a document to fill the computer screen. The
> view hides menus, toolbars, and taskbars.
>
> <span id="gt_19363950-99e9-4f11-a562-96e4dd4ea5ce"
> class="anchor"></span>**grammar checker**: An application that uses
> default or user-defined settings to search for grammatical errors in a
> document.
>
> <span id="gt_69c5a114-8e6b-4003-b8e4-a06577cfc226"
> class="anchor"></span>**grammar checker cookie**: An entity in a
> document that a grammar checker uses to denote a possible grammatical
> error in the document and data about that error.
>
> <span id="gt_d38255f8-dca6-4e1c-be9d-71691a6ab4e5"
> class="anchor"></span>**gutter**: An area above a column heading and
> to the left of a row heading. A gutter typically displays outline
> symbols that are used to expand and collapse groups of cells.
>
> <span id="gt_384fedff-6be0-4cc3-9361-0430ba5ec2dd"
> class="anchor"></span>**gutter margin**: A margin setting that adds
> extra space to the side or top margin of a document that will be
> printed and bound. A gutter margin ensures that text is not obscured
> by the binding.
>
> <span id="gt_9ba05db2-b238-4ee0-a0b6-93a436452e68"
> class="anchor"></span>**Hangul-Hanja converter (HHC)**: A collection
> of dictionaries that readers can use to search for and select a Hanja
> word that corresponds to a specified Hangul word, or a Hangul word
> that corresponds to a specified Hanja word.
>
> <span id="gt_84a82291-9444-481e-97e3-4ff69a88e932"
> class="anchor"></span>**header**: A line, or lines, of content in the
> top margin area of a page in a document or a slide in a presentation.
> A header typically contains elements such as the title of the chapter,
> the title of the document, a page number, or the name of the author.
>
> <span id="gt_e68269e4-278c-4bfa-a888-d3fce7942c3c"
> class="anchor"></span>**heading style**: A type of paragraph style
> that also specifies a heading level. There are as many as nine
> built-in heading styles, Heading 1 through Heading 9.
>
> <span id="gt_29e4e1ad-bba7-40a0-81f6-f7ad1df73cf4"
> class="anchor"></span>**horizontal band**: A set of rows in a table
> that are treated as a single unit, typically to ensure the consistency
> of the layout and the format.
>
> <span id="gt_e6f7b5be-1735-4258-807a-812a1e91e611"
> class="anchor"></span>**HTML image map**: An image that contains more
> than one hyperlink on a webpage. Clicking various parts of the image
> links the user to other resources on another part of the page, a
> different page, or a file.
>
> <span id="gt_eaceec1b-55ef-419c-b036-2dbb4958a113"
> class="anchor"></span>**hybrid list**: A nine-level list that is
> exposed in the user interface as a collection of nine, one-level
> lists, instead of a single nine-level list.
>
> <span id="gt_5eb8210a-2e2b-4c00-b7ce-73fd3e8720cb"
> class="anchor"></span>**Hyperlink view**: A document view that
> displays a document as it would appear as a webpage.
>
> <span id="gt_549c4960-e8be-4c24-bc2b-b86530f1c1bf"
> class="anchor"></span>**Hypertext Markup Language (HTML)**: An
> application of the Standard Generalized Markup Language (SGML) that
> uses tags to mark elements in a document, as described in
> [\[HTML\]](https://go.microsoft.com/fwlink/?LinkId=89880).
>
> <span id="gt_1aa4deeb-8e50-4699-8e40-2c0719250cbb"
> class="anchor"></span>**incremental save**: A process in which an
> existing file is modified to reflect only additions or changes to a
> document, while maintaining all other existing content in the file.
>
> <span id="gt_19d3dd70-8d6a-4847-9827-ebbde6b1a9b7"
> class="anchor"></span>**Input Method Editor (IME)**: An application
> that is used to enter characters in written Asian languages by using a
> standard 101-key keyboard. An IME consists of both an engine that
> converts keystrokes into phonetic and ideographic characters and a
> dictionary of commonly used ideographic words.
>
> <span id="gt_fad7dafb-1cf1-455e-820b-e2b34586a6f3"
> class="anchor"></span>**insertion point**: A position between two
> existing characters, or a position before or after a character, where
> text can be inserted. If a caret is positioned at an insertion point,
> the point can have unique formatting, which is applied to any text
> that is inserted at the insertion point.
>
> <span id="gt_3ec90cfa-5a13-4285-9296-70340122d2eb"
> class="anchor"></span>**kinsoku**: A rule set in the Japanese language
> that is used to determine characters that are not permitted at the
> beginning or end of a line.
>
> <span id="gt_ab922a45-66c3-4368-9c3f-500e2dc1c6e7"
> class="anchor"></span>**Kumimoji**: A text layout setting that
> displays annotative characters inline next to the text to which they
> apply. It is typically used with East Asian text to indicate
> pronunciation.
>
> <span id="gt_242f86a5-2dc4-4ef3-8d4a-d1b5a7aed0aa"
> class="anchor"></span>**labels document**: A document that stores
> label design and printing information in conjunction with a mail merge
> document.
>
> <span id="gt_a75c76d8-cf01-4540-a8e7-24b367b28370"
> class="anchor"></span>**language auto-detection**: A process that
> automatically determines the language code identifier (LCID) for text
> in a document.
>
> <span id="gt_b8c262f8-6c09-457a-9b68-4bf3a08ab067"
> class="anchor"></span>**left-to-right**: A reading order in which
> characters in words are read from left to right, and words are read
> from left to right in sentences.
>
> <span id="gt_4093a4dd-d31a-4787-b0e7-a55ac524f588"
> class="anchor"></span>**line numbers**: A formatting property in which
